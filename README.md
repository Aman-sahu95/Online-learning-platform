📚 Online Learning Platform (Biotech)

🔗 Live Demo: https://online-learning-platform-5-u1k1.onrender.com

🚀 Overview

This is a Biotech-based Online Learning Platform developed using HTML, CSS, JavaScript (Frontend) and Python Django (Backend).
The platform allows students to explore biotechnology courses, interact with the system, and communicate with the admin.

✨ Features

👤 User Features

User Registration & Login System
Access Biotech Courses
View learning content
Send Enquiries
Receive updates via Email

🎓 Student Panel

View available courses
Manage personal profile
Interact with platform

🛠️ Admin Panel

Add / Update / Delete courses
Manage users
Handle enquiries
Monitor platform activities
📩 Enquiry & Email System
Users can submit queries
Admin can respond
Email notifications for communication
🛠️ Tech Stack

Frontend: HTML,CSS,JavaScript

Backend: Python wiyh Django
Database: SQLite
